## DataMatchingAssistant:

**DataMatchingAssistant** is a Python tool that automates data reconciliation between two Excel files. Using advanced string matching algorithms, it ensures data accuracy and quality, streamlining the comparison process.

### Features:
- **Graphical User Interface**: Intuitive interface built with Tkinter for easy file and column selection.
- **Pandas Integration**: Efficient data import and manipulation from Excel files.
- **Advanced Matching with Fuzzywuzzy**: Matches records based on string similarity.

A practical tool for those handling large datasets, **DataMatchingAssistant** simplifies data reconciliation tasks.
